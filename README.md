## Description

Mountain JourneyMountainJourney is the new web application for create manage and share your hiking trails for your next trip with a powerfull engine in 3D. MJ have a API for manage users information and trails map, and use the frontend API of Mapbox for create the map.

This is a project free of right. You can use it has you wish if you integre my names. Thanks for your reading.
